###  Trabalho 2 — CRUD de Biblioteca em Dart (POO)
**Responsável:** Ramon
**Branch:** [`dev-ramon`](https://github.com/jpktolentino-dot/Atividades-Avaliativas-Mobile/tree/dev-ramon)
**Valor:** 2,5 pontos

Sistema de gerenciamento de biblioteca executado no terminal, desenvolvido com **Programação Orientada a Objetos em Dart**. Permite cadastrar, listar, atualizar e remover livros via menu interativo.

Conceitos aplicados: classes, atributos, métodos, construtores e listas.

---

###  Trabalho 3 — Calculadora em Flutter com Componentização
**Responsável:** João
**Branch:** [`dev-joão`](https://github.com/jpktolentino-dot/Atividades-Avaliativas-Mobile/tree/dev-jo%C3%A3o)
**Valor:** 2,5 pontos

Aplicativo de calculadora simples desenvolvido em **Flutter**, com foco em **componentização de widgets**. Suporta as quatro operações matemáticas básicas e organiza a interface em widgets reutilizáveis (botões numéricos, botões de operação, display).

###  Trabalho 4 — Lista de Tarefas com Gestão de Estado (Riverpod)
**Responsável:** Eduardo
**Branch:** [`dev-eduardo`](https://github.com/jpktolentino-dot/Atividades-Avaliativas-Mobile/tree/dev-eduardo)
**Valor:** 2,5 pontos

Aplicativo de **To-Do List** desenvolvido em Flutter com gestão de estado via **Riverpod**. Permite adicionar, visualizar, marcar como concluídas e remover tarefas, com indicação visual do status de cada item.
