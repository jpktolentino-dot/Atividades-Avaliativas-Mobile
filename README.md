# 📖 App de lista de tarefas em dart

Este projeto consiste em um sistema simples de lista de tarefas desenvolvido em **Dart**

------------------------------------------------------------------------

## 🎯 Objetivo

-   Aplicar conceitos de gestão de estado em dart
-   Desenvolver um sistema interativo 
-   Implementar operações criar tarefa, mudar status e deletar tarefa

------------------------------------------------------------------------

## 🛠️ Tecnologias Utilizadas

-   **Dart**
-   Biblioteca padrão `dart:io` para entrada e saída de dados
-   Biblioteca padrão `riverpod` para entrada e saída de dados

------------------------------------------------------------------------

## 📌 Funcionalidades

O sistema permite:

- Criar tarefa
- Visualizar tarefa
- Mudar status da uma tarefa de a fazer para feita
- Deletar tarefa

------------------------------------------------------------------------


## ⚙️ Instalação do Sistema

- Certifique-se de ter o flutter instalado
- Certifique-se de ter o riverpod instalado

```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.5.1 # Ou a versão mais recente
```
- Clone o repositório

```bash
git clone https://github.com/jpktolentino-dot/Atividades-Avaliativas-Mobile/new/dev-eduardo
```

- Rode o código
- O sistema gerará uma página no navegador
------------------------------------------------------------------------

## 💻 Exemplo de Uso
- O usuário digita uma tarefa no campo de texto
- O usuário Clica no botão + para adicionar
- O usuário pode clicar na caixa ao lado esquerdo para marcar a tarefa como feita e pode deletar clicando no icone de lixeira no lado direito
------------------------------------------------------------------------

## ⚠️ Observações

-   Os dados são armazenados apenas em memória (lista), ou seja, não há
    persistência em banco de dados\
